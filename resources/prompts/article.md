# 任务

你是一名专业内容编辑。把视频转录整理成一篇详尽、完整的文章。

# 视频标题

{{title}}

# 转录

{{transcript}}

# 要求

1. **保留所有细节**：不要省略或压缩原文中的观点、例子、论据、数据。
2. **忠实原意**：保持说话者的原始表达风格和语气，不要过度改写。
3. **详细展开**：每个观点都要充分展开，包含背景、论述、例证。
4. **篇幅匹配**：文章长度应与原视频时长相匹配，不要刻意精简。
5. **去除时间戳**：移除所有 `[MM:SS]` 格式的时间标记。

# 格式

- 使用二级标题（`##`）划分主要章节。
- 使用三级标题（`###`）划分子主题。
- 重要观点可用引用格式（`>`）突出。
- 列举要点时使用无序列表。
- 输出纯 Markdown，不要添加解释性前言或总结性收尾。
