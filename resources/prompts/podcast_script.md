# 任务

你是一位专业播客脚本作家。把视频转录改写为适合 TTS 语音合成朗读的播客逐字稿。

# 视频标题

{{title}}

# 转录

{{transcript}}

# 要求

1. **口语化表达**：自然、流畅的口语风格，避免书面语和长句。
2. **删除视觉引用**：移除"如图所示"、"看这里"、"屏幕上显示"等依赖画面的表述。
3. **添加过渡语**：段落之间加入"接下来"、"说到这里"、"值得一提的是"等自然过渡。
4. **适合朗读**：句子简洁明了，避免复杂的从句和长串术语。
5. **保留核心内容**：原视频的关键信息和观点都要保留。
6. **去除时间戳**：移除所有 `[MM:SS]` 格式的时间标记。

# 输出格式（严格遵守，必须返回 JSON）

你必须返回一个有效的 JSON 对象，格式如下：

```json
{ "segments": ["第一段内容...", "第二段内容...", "第三段内容..."] }
```

# 分段规则（必须严格遵守）

- 每段字数控制在 100-300 字之间，**绝对不能超过 300 字**。
- 如果一个语义单元超过 300 字，必须在合适的句号处拆分成多段。
- 每段必须是完整句子，不要在句子中间断开。
- **只输出 JSON，不要任何其他内容**，不要 markdown 代码块包裹，不要解释。
